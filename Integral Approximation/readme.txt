This program will approximate the value of integral for inputted value
How to compile : g++ -o main integral_approx.cpp
How to use :
1. Input the number of polynom you want
2. Enter all of the coefficient and degree
3. Enter the interval
4. Enter how precise you want (all in lower case and no blank space)
 example : low
